# <a name="set-up-and-use-pay-when-paid-vendor-payments"></a><span data-ttu-id="a56cb-103">Konfiguratu eta erabili ordaindu ordaintzean saltzailearen ordainketak</span><span class="sxs-lookup"><span data-stu-id="a56cb-103">Set up and use pay-when-paid vendor payments</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="a56cb-104">Saltzaile bat azpikontratatu gisa lan egitea onartzen duzunean, saltzaileari ordainketa gorde nahi diozu, bezeroak proiektua ordaindu arte.</span><span class="sxs-lookup"><span data-stu-id="a56cb-104">When you approve a vendor to work as a subcontractor, you might want to withhold payment to the vendor until your customer pays you for the project.</span></span> <span data-ttu-id="a56cb-105">Eszenatoki hau onartzeko, ordaindutakoan (PWP) baldintzak konfigura ditzakezu saltzailearekin erosketa eskaera (PO) konfiguratzen duzunean.</span><span class="sxs-lookup"><span data-stu-id="a56cb-105">To support this scenario, you can set up pay-when-paid (PWP) terms when you set up the purchase order (PO) with the vendor.</span></span>

<span data-ttu-id="a56cb-106">Adibidez, bezeroak proiektuaren fakturan zenbateko bat ordaintzen duenean, saltzailearen fakturaren zenbateko bat edo guztia askatu dezakezu.</span><span class="sxs-lookup"><span data-stu-id="a56cb-106">For example, when a customer pays an amount on a project invoice, you can release some or all of the vendor invoice amount.</span></span> <span data-ttu-id="a56cb-107">Just konfiguratu PWP baldintzak zehazten duten saltzaileari ordainduko dela bezeroarekin erlazionatutako ordainketaren ehuneko bat jaso ondoren.</span><span class="sxs-lookup"><span data-stu-id="a56cb-107">Just set up PWP terms that specify that the vendor will be paid after you receive a percentage of the related payment from the customer.</span></span> <span data-ttu-id="a56cb-108">Bezero baten ordainketa partziala jasotzen baduzu, erlazionatutako saltzailearen faktura batzuk eskuz askatu ditzakezu ordainketa egiteko.</span><span class="sxs-lookup"><span data-stu-id="a56cb-108">If you receive partial payment from a customer, you can manually release some of the related vendor invoices for payment.</span></span>

<span data-ttu-id="a56cb-109">Hurrengo adibidean PWP terminoak erabiltzen direnean prozesua azaltzen da.</span><span class="sxs-lookup"><span data-stu-id="a56cb-109">The following example outlines the process when PWP terms are used.</span></span>

## <a name="example"></a><span data-ttu-id="a56cb-110">Adibidez</span><span class="sxs-lookup"><span data-stu-id="a56cb-110">Example</span></span>

<span data-ttu-id="a56cb-111">Zure erakundeak onartzen du bezeroari softwarea instalatuta duten 100 ordenagailu eskaintzea, ordenagailu bakoitzeko 150,00 dolar (USD) prezioan.</span><span class="sxs-lookup"><span data-stu-id="a56cb-111">Your organization agrees to provide a customer with 100 computers that have software installed, for a price of 150.00 US dollars (USD) per computer.</span></span> <span data-ttu-id="a56cb-112">Saltzaile bat kontratatuko duzu softwarea instalatuta duten ordenagailuak hornitzeko.</span><span class="sxs-lookup"><span data-stu-id="a56cb-112">You then hire a vendor to provide you with the computers that have software installed.</span></span> <span data-ttu-id="a56cb-113">Hitzarmenaren arabera, bezeroak zure erakundeari ordaindu aurretik ordenagailuen kalitatea onartu behar du.</span><span class="sxs-lookup"><span data-stu-id="a56cb-113">According to the agreement, the customer must approve the quality of the computers before it pays your organization.</span></span> <span data-ttu-id="a56cb-114">Zure erakundearen gidalerroa saltzaileei ordainketa atxikitzea da, bezeroak ordaindu arte.</span><span class="sxs-lookup"><span data-stu-id="a56cb-114">Your organization's policy is to withhold payment to vendors until the customer has paid you.</span></span> <span data-ttu-id="a56cb-115">Hori dela eta, proiektua % 100eko PWP ehunekoa izan dezan konfiguratzen duzu.</span><span class="sxs-lookup"><span data-stu-id="a56cb-115">Therefore, you set up the project so that it has a PWP percentage of 100 percent.</span></span>

<span data-ttu-id="a56cb-120">Bezeroaren ordainketa osoa jaso ondoren, saltzaileari 10.000,00 ordainduko diozu, saltzailearen fakturaren zenbateko osoa.</span><span class="sxs-lookup"><span data-stu-id="a56cb-120">After you receive the full payment from the customer, you pay the vendor 10,000.00, the full amount of the vendor invoice.</span></span>

    - <span data-ttu-id="a56cb-129">**Taula** : PWP baldintzak saltzaile bakar bati aplikatzen zaizkio.</span><span class="sxs-lookup"><span data-stu-id="a56cb-129">**Table** – The PWP terms apply to a single vendor.</span></span>
